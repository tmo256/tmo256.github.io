HTML Templates

HTML templates coded with Flexbox and CSS Grid methods of layout are available in this repository (Flexbox Layout and CSS Grid Layout). Obviously, the technical environment of any specific application may prohibit the direct implementation of that HTML; use the specifications provided here to begin a custom implementation of the UL layout if necessary.

As described elsewhere in this documentation, the font family is Open Sans; the font weight is 400. Also as specified in Typography, the rem unit is assumed to be 16px when body font size is set to 100% (or explicitly set to 16). Because moving smoothly between applications is the goal of a common header, pixels may be the preferred unit when coding your header element (both rems and pixels are provided here).

A note on the submenus provided in templates: the style, content and animation of menus triggered by the account and menu icons may vary from application to application. If possible, transition the appearance of the menus in a similar style and from the same direction as the template indicate; however, the conventions and limitations of individual applications may supersede the method illustrated here.
