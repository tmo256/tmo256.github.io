Tabs

Use tabs to break up related information within the same context. However, using tabs can present multiple challenges in responsive situations, especially in mobile-sized views. These problems include inadequate horizontal space to display tab labels (especially when translated into multiple languages, where text width will be unknown), fluctuating content element heights (affecting layout as the tab changes), difficulty with internal scrolling of content, etc. Avoid where possible.

If tabs are necessary and present problems in smaller responsive views, display all content. If the content is exceptionally long, maintain tab links as a menu of jump links and include a “Return to top” link back to the submenu.
