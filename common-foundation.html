Common Foundation

Implementing the common foundation in an application is intended to accomplish 80% of a common look and should (and is intended to) be able to be accomplished quickly.

  • Header color and UL logo. UL red header is preferred but for historical / persona reasons other choices are acceptable- Good Guide, for example, serves an entirely different persona from what is more typical for other UL apps.
  • Footer color. Dark grey, common link styles and elements (copyright, etc.) where possible.
  • Font: Adopt Open Sans for header tags, at minimum.
  • Icons: Font Awesome (there is some cost associated with this).
  • Button styles: Padding / border radius, shape, border, shadow, mouseovers (transitions, relative lightness/brightness). Not necessary, color or typography.
